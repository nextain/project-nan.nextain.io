# 로고 변경 및 메인 히어로 섹션 배경 개선 계획

## 1. 개요
- 사용자가 제공한 새로운 `public/logo.jpg` 로고를 적용한다.
- 다소 밋밋한 메인 페이지 히어로 섹션의 배경에 시각적 효과를 추가한다.

## 2. 작업 계획

### 2.1. 로고 변경
- **대상 파일**: `src/components/layout/header.tsx`
- **수정 내용**:
  - 기존의 `lucide-react` `<Coffee>` 아이콘을 제거한다.
  - `next/image` 컴포넌트를 사용하여 `public/logo.jpg`를 렌더링하도록 수정한다.
  - 로고의 크기가 헤더에 어울리도록 적절한 `width`와 `height` 값을 설정한다.

### 2.2. 히어로 섹션 배경 개선
- **대상 파일**: `src/components/home/hero.tsx`
- **개선 방안**: `public/globe.svg`를 활용한 미세한 회전 애니메이션 추가
  - 기존 그라데이션 배경은 유지한다.
  - `globe.svg` 이미지를 배경에 `position: absolute`로 크게 배치한다.
  - `opacity`를 낮춰 은은하게 보이도록 처리한다.
  - CSS `@keyframes`를 사용하여 이미지가 아주 느리게 회전하는 애니메이션을 적용한다.
  - 이 방법은 추가 이미지 리소스 없이 기존 자원을 활용하여 동적이고 세련된 느낌을 줄 수 있다.

## 3. 예상 결과
- 웹사이트 헤더에 새로운 로고가 정상적으로 표시된다.
- 메인 페이지 접속 시, 정적인 배경 대신 미세하게 움직이는 지구본 이미지가 추가되어 시각적인 만족도를 높인다.
