Alpha와 대화하는 메인 화면입니다.

## 텍스트 채팅

![채팅 화면](chat-text.png)

1. 하단 입력창에 메시지를 입력합니다
2. **Enter** 키 또는 **전송 버튼**을 클릭합니다
3. Alpha가 실시간으로 응답을 생성합니다 (스트리밍)
4. 줄바꿈은 Shift+Enter를 사용합니다

## 음성 채팅

1. 입력창 옆의 **마이크 버튼**을 클릭합니다
2. 마이크가 활성화되면 말합니다
3. 다시 클릭하면 녹음 중지 — 음성이 텍스트로 변환됩니다 (STT)
4. 변환된 텍스트가 자동으로 전송됩니다

## 음성 응답 (TTS)

- Alpha의 응답이 자동으로 음성으로 재생됩니다
- 설정에서 끌 수 있습니다
- 재생 중 아바타가 립싱크 애니메이션을 합니다

## 비용 표시

![비용 대시보드](chat-cost.png)

- 우측 상단에 현재 세션의 **API 비용 합계**가 표시됩니다
- 클릭하면 상세 비용 대시보드가 열립니다:
  - 제공업체별 메시지 수
  - 입력/출력 토큰 수
  - Lab 잔액 (연결 시)
  - 크레딧 충전 링크

## 새 대화

- 우측 상단 **+** 버튼을 클릭하면 새 대화를 시작합니다
- 이전 대화는 기록 탭에 저장됩니다

## 도구 실행 표시

Alpha가 도구를 사용하면 채팅 영역에 표시됩니다:

![도구 실행 표시](chat-tool.png)

- 도구 이름 (예: "파일 읽기", "명령 실행")
- 실행 상태 (실행 중 / 성공 / 오류)
- 결과 (펼쳐서 확인 가능)

## 승인 모달

보안이 필요한 도구 실행 시 승인 모달이 나타납니다:

![승인 모달](chat-approval.png)

| 버튼 | 설명 |
|------|------|
| **한 번 허용** | 이번 실행만 허용 |
| **항상 허용** | 이 도구를 앞으로 자동 승인 |
| **거부** | 실행을 거부 |
