## 아바타

### VRM 모델
- 4가지 내장 모델 또는 **파일 선택**으로 커스텀 VRM을 불러올 수 있습니다
- 클릭하면 즉시 미리보기됩니다
- 저장하지 않고 나가면 이전 모델로 되돌아갑니다

### 배경 이미지
- 기본 그라디언트 또는 내장 배경
- **파일 선택**으로 커스텀 이미지를 불러올 수 있습니다 (PNG, JPG, WebP)

## Cafelua Lab 계정

Lab과의 연동을 관리합니다.

### 연결된 상태

- 사용자 ID 표시
- **크레딧 잔액** 표시 (실시간 조회)
- **대시보드** 버튼: Lab 대시보드를 브라우저에서 열기
- **크레딧 충전** 버튼: Lab 결제 페이지로 이동
- **연결 해제** 버튼: Lab 계정 연결 해제
- 설정 저장 시 Lab에 자동 동기화

### 미연결 상태
- **Lab 로그인** 버튼: 브라우저로 로그인 후 자동 연결

## 사용법 매뉴얼

"사용법 매뉴얼" 클릭 시 이 페이지가 브라우저에서 열립니다.

## AI 설정

- **제공업체**: Gemini (기본), OpenAI, Claude, Grok, zAI, Ollama
  - 현재 Gemini만 사용 가능; 나머지는 추후 지원 예정
- **모델**: 모델 이름 (예: gemini-2.5-flash)
- **API 키**: 제공업체 API 키 (Lab 사용 시 불필요)

## 음성 (TTS/STT)

- **음성 응답 (TTS)**: ON/OFF
- **음성 입력 (STT)**: ON/OFF
- **Google API 키**: 전용 TTS/STT 키 (비우면 채팅 키 사용)
- **TTS 음성**: 11가지 한국어 음성 선택
  - Neural2: 높은 품질 ($16/100만 자)
  - WaveNet: 자연스러운 ($16/100만 자)
  - Standard: 기본 ($4/100만 자)
- **미리듣기** 버튼으로 선택한 음성을 테스트

## 도구

AI 아바타가 사용할 수 있는 시스템 제어 권한을 관리합니다.

- **도구 활성화**: 도구 기능 마스터 ON/OFF
- **Gateway URL / Gateway Token**: 일반 사용자 환경에서는 자동으로 관리되며 직접 입력할 필요가 없습니다
  - 자체 Gateway를 운영하는 고급 사용자만 수동으로 확인/수정합니다
- **허용된 도구**: "항상 허용"으로 설정된 도구 목록
  - **허용 도구 초기화**: 모든 승인 초기화

> **보안 등급(T0~T3) 안내**
> AI가 도구를 사용할 때 시스템에 미치는 영향에 따라 4단계 보안 등급이 적용됩니다.
> - **T0 (읽기 전용)**: 조회 작업. 사용자 개입 없이 자동 실행됩니다.
> - **T1 (알림)**: 단순 외부 검색/알림 등. 실행 시 화면에 알림만 표시됩니다.
> - **T2 (주의)**: 파일 생성/수정 등 시스템 변경 작업. 사용자의 명시적 승인이 필요합니다.
> - **T3 (위험)**: 터미널 명령어 실행 등 위험 작업. 매회 승인이 필수입니다.

## 채널 (Channels)

- **채널 관리**: 현재 연결된 메신저 채널의 상태를 확인하고, 외부 서비스와의 연동을 제어합니다.
- (상세한 제어는 채널 탭에서 수행합니다)

## 연동 (Integrations)

**설정 > 연동** 페이지에서 외부 메신저 서비스와의 연동 상태를 확인하고 관리합니다.

- **Discord 연동**: 디스코드 계정으로 로그인하면 자동 연동됩니다. 봇 초대 링크로 서버에 봇을 추가하면 멘션이나 DM으로 AI와 대화할 수 있습니다.
- **Google Chat 연동**: 구글 계정으로 로그인하면 Google Chat 앱을 통해 대화할 수 있습니다.
- 연동 상태(연결됨/미연결)가 배지로 표시됩니다.

## 기기 (Device) 및 호출어

![기기 및 호출어 설정](settings-device.png)

- **호출어 목록**: AI를 깨울 때 사용할 단어를 관리합니다 (예: "알파야", "안녕 알파")
- **디바이스 페어링**: 외부 기기 연동 (예: 페어링된 OpenClaw 노드) 및 스마트 홈 기기와의 연결 상태를 관리합니다.
- 기기 승인 및 토큰 리셋 기능이 포함됩니다.

## 메모리 (학습된 사실)

AI 아바타가 대화에서 학습한 사실을 관리합니다.
- 각 사실은 **키**와 **값**으로 표시됩니다
- **삭제** 버튼으로 불필요한 사실을 제거할 수 있습니다
- 저장된 사실은 다음 대화의 응답 품질에 영향을 줍니다
- 사실을 많이 지우면 개인화 수준이 낮아질 수 있습니다

## 저장 및 초기화

- **저장**: 모든 변경 사항 적용
- **전체 초기화**: 모든 설정과 카메라를 기본값으로 복원 (확인 필요)
