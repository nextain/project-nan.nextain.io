자주 발생하는 문제와 해결 방법입니다.

## AI 에이전트가 응답하지 않아요

1. 설정 > AI 설정에서 API 키 또는 Lab 연결을 확인하세요
2. 인터넷 연결을 확인하세요
3. 설정 > Cafelua Lab 계정에서 크레딧 잔액을 확인하세요
4. 모델이 올바르게 설정되어 있는지 확인하세요

## 음성이 작동하지 않아요

1. 설정 > 음성에서 TTS/STT가 활성화되어 있는지 확인하세요
2. Google API 키가 필요할 수 있습니다
3. 마이크 권한을 확인하세요 (OS 설정)
4. 브라우저 미디어 권한이 차단되어 있지 않은지 확인하세요

## 크레딧이 부족해요

1. 설정 > Cafelua Lab 계정에서 잔액을 확인하세요
2. Lab 대시보드에서 사용량을 확인하세요
3. 무료 크레딧은 매월 충전됩니다
4. BASIC ($10/월)으로 업그레이드하면 100 크레딧을 받습니다

## 도구가 작동하지 않아요

1. 설정 > 도구에서 "도구 활성화"가 켜져 있는지 확인하세요
2. Gateway가 실행 중인지 확인하세요
3. 스킬 탭에서 필요한 스킬이 활성화되어 있는지 확인하세요
4. 일반 사용자 환경에서는 Gateway URL/토큰을 직접 입력할 필요가 없습니다
5. 자체 Gateway를 운영하는 경우에만 Gateway URL/토큰을 확인하세요

## 앱이 느리거나 멈춰요

1. 새 대화 시작(+) — 긴 대화는 느려질 수 있습니다
2. 앱을 재시작하세요
3. 설정 > 전체 초기화로 완전히 리셋하세요

## 날씨가 안 나와요

- 날씨 스킬은 인터넷 연결이 필요합니다
- "서울 날씨 알려줘"처럼 구체적으로 요청하세요
- 영문 도시명으로도 시도해보세요 (예: "Seoul weather")

## Lab 로그인이 안 돼요

1. 인터넷 연결을 확인하세요
2. 브라우저에서 lab.cafelua.com에 직접 접속해보세요
3. Google 계정으로 로그인이 되는지 확인하세요
4. 앱을 재시작 후 다시 시도하세요
