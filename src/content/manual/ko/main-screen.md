앱의 메인 화면은 두 영역으로 나뉩니다.

![메인 화면 레이아웃](main-screen.png)

## 아바타 영역 (위쪽)

- 선택한 VRM 캐릭터가 3D로 렌더링됩니다
- **감정 표현**: 대화에 따라 표정이 변합니다
  - [HAPPY] — 기쁨
  - [SAD] — 슬픔
  - [ANGRY] — 화남
  - [SURPRISED] — 놀람
  - [NEUTRAL] — 중립
  - [THINK] — 생각 중
- **카메라 조작**: 아바타 영역을 드래그해 시점을 이동할 수 있습니다
- **배경 이미지**: 아바타 뒤에 배경이 표시됩니다
- **립싱크**: TTS 응답 시 입모양 애니메이션이 재생됩니다

## 패널 영역 (아래쪽)

아래 패널에는 5개 탭이 있습니다:

| 탭 | 설명 |
|-----|------|
| 채팅 | Alpha와 대화 |
| 기록 | 이전 대화 목록 |
| 작업 | 도구 실행/오류 모니터링 |
| 스킬 | 사용 가능한 스킬(도구) 목록 |
| 설정 | 모든 앱 설정 |

각 탭의 상세 기능은 해당 페이지에서 확인하세요.
