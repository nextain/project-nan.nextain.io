AI의 작업 활동을 모니터링하는 탭입니다.

![작업 탭](progress-tab.png)

## 통계 요약

상단에 4가지 통계가 표시됩니다:

| 항목 | 설명 |
|------|------|
| **총 이벤트** | 전체 작업 수 |
| **총 비용** | 사용한 API 비용 |
| **도구 사용** | 도구 호출 횟수 |
| **오류** | 발생한 오류 횟수 |

## 이벤트 목록

- 각 이벤트의 타입, 시간, 상세 내용이 표시됩니다
- **새로고침** 버튼으로 최신 데이터를 확인합니다
- 오류만 필터링하여 볼 수 있습니다
