## Resumen de Estadísticas

Cuatro estadísticas en la parte superior:

| Elemento | Descripción |
|-----------|-------------|
| **Eventos Totales** | Número total de operaciones |
| **Costo Total** | Costos de API utilizados |
| **Herramientas Utilizadas** | Número de herramientas invocadas |
| **Errores** | Número de errores encontrados |

## Lista de Eventos

- Cada evento muestra tipo, marca de tiempo y detalles
- Botón **Actualizar** para los datos más recientes
- Filtrar para mostrar solo errores
