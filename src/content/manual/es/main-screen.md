La pantalla principal tiene dos áreas.

![Diseño de la pantalla principal](main-screen.png)

## Área del Avatar (Superior)

- Tu personaje VRM elegido se renderiza en 3D
- Las **emociones** cambian según la conversación:
  - [HAPPY] — Alegría
  - [SAD] — Tristeza
  - [ANGRY] — Enojo
  - [SURPRISED] — Sorpresa
  - [NEUTRAL] — Neutral
  - [THINK] — Pensando
- **Control de cámara**: Arrastra en el área del avatar para mover el punto de vista
- **Imagen de fondo**: Aparece detrás del avatar
- **Sincronización labial**: La animación se reproduce durante las respuestas TTS

## Área del Panel (Inferior)

El panel inferior tiene 8 pestañas:

| Pestaña | Descripción |
|---------|-------------|
| Chat | Conversación con AI 아바타 |
| Historial | Conversaciones anteriores |
| Progreso | Monitorización de ejecución/errores de herramientas |
| Habilidades | Lista de habilidades (herramientas) disponibles |
| Canales | Gestionar canales de mensajería conectados |
| Agentes | Agentes en segundo plano y gestión de sesiones |
| Diagnóstico | Estado del sistema de la pasarela y registros |
| Ajustes | Todas las configuraciones de la aplicación |

Consulta la página de cada pestaña para ver las características detalladas.
