一般的な問題と解決策。

## AIエージェントが応答しない

1. [設定] > [AI設定] でAPIキーまたはLab接続を確認する
2. インターネット接続を確認する
3. [設定] > [Labアカウント] でクレジット残高を確認する
4. モデルが正しく設定されていることを確認する

## 音声が動作しない

1. [設定] > [音声] でTTS/STTが有効になっていることを確認する
2. Google API keyが必要な場合があります
3. マイクの許可設定を確認する (OS設定)
4. ブラウザのメディア許可設定がブロックされていないことを確認する

## クレジットが不足している

1. [設定] > [Labアカウント] で残高を確認する
2. Lab Dashboardで使用状況を確認する
3. 無料クレジットは毎月補充されます
4. 100クレジットのためにBASIC (月額$10) にアップグレードする

## ツールが動作しない

1. [設定] > [ツール] で「ツールを有効にする」がオンになっていることを確認する
2. Gatewayが実行されていることを確認する
3. スキルタブで必要なスキルが有効になっていることを確認する
4. 通常のユーザー環境では、Gateway URL/tokenを手動で入力する必要はありません
5. Gateway URL/tokenは、自己ホスト型/高度なセットアップの場合のみ確認してください

## アプリが遅い、またはフリーズする

1. 新しい会話を開始する (+) — 長いチャットは速度を低下させる可能性があります
2. アプリを再起動する
3. 完全にリセットするには、[設定] > [すべてリセット] を使用する

## 天気が表示されない

- 天気スキルにはインターネット接続が必要です
- 具体的に指定してください：「ソウルの天気を教えて」
- 英語の都市名で試してください (例: "Seoul weather")

## Labログインが動作しない

1. インターネット接続を確認する
2. ブラウザで直接 naia.nextain.io にアクセスしてみてください
3. Googleアカウントでサインインできることを確認する
4. アプリを再起動して、もう一度試す
