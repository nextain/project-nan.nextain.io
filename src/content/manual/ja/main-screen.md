メイン画面は2つの領域に分かれています。

![Main screen layout](main-screen.png)

## アバター領域（上部）

- 選択したVRMキャラクターが3Dで表示されます
- 会話に基づいて**感情**が変化します:
  - [HAPPY] — 喜び
  - [SAD] — 悲しみ
  - [ANGRY] — 怒り
  - [SURPRISED] — 驚き
  - [NEUTRAL] — 通常
  - [THINK] — 思考
- **カメラ操作**: アバター領域をドラッグして視点を移動します
- **背景画像**: アバターの後ろに表示されます
- **リップシンク**: TTS応答中にアニメーションが再生されます

## パネル領域（下部）

下部パネルには8つのタブがあります:

| タブ | 説明 |
|-----|-------------|
| チャット | AI アバターとの会話 |
| 履歴 | 過去の会話 |
| 進行状況 | ツールの実行/エラー監視 |
| スキル | 利用可能なスキル（ツール）の一覧 |
| チャンネル | 接続されているメッセンジャーチャンネルの管理 |
| エージェント | バックグラウンドエージェントとセッション管理 |
| 診断 | ゲートウェイシステムの健全性とログ |
| 設定 | アプリのすべての設定 |

詳細な機能については、各タブのページを参照してください。
