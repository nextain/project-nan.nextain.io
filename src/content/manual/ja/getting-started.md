アプリを初めて起動すると、オンボーディングウィザードが表示されます。開始するには2つの方法があります。

## Naia OSで開始する（推奨）

最も簡単な方法 — 無料でAPI Keyは不要です。

![Onboarding screen — provider selection](onboarding-provider.png)

1. オンボーディング画面で**Naia OS**ボタンをクリックします
2. ブラウザがnaia.nextain.ioのログインページを開きます
3. **Googleアカウント**でサインインします（登録は自動です）
4. デスクトップアプリに自動的にリダイレクトされます
5. サインアップ時に**20無料クレジット**を受け取ります

## API Keyで開始する

Google GeminiのようなプロバイダーからAPI Keyを入力することもできます。

![Onboarding screen — API key entry](onboarding-apikey.png)

1. プロバイダーを選択します（Geminiなど）
2. **次へ**をクリックします
3. API Keyを入力します
4. **検証**をクリックしてキーをテストします

## オンボーディング手順

ラボログインまたはAPI Key入力後、以下の手順を完了してください。

### 1. AIに名前を付ける
AI 아바타に好きな名前を付けてください（例：「AI 아바타」、「Luna」、「Cookie」）。

![AI name input](onboarding-agent-name.png)

### 2. あなたの名前
AI 아바타に呼ばれたい名前を入力してください。

![Your name input](onboarding-user-name.png)

### 3. キャラクターを選択する
4つのVRMアバターから選択してください。

| キャラクター | 説明 |
|-----------|-------------|
| シノ（ダーク） | ダークユニフォーム |
| シノ（ライト） | ライトユニフォーム |
| ガール | 一般的な女の子 |
| ボーイ | 一般的な男の子 |

![Character selection screen](onboarding-character.png)

### 4. パーソナリティを選択する
4つのプリセットから選択してください。

| パーソナリティ | 説明 |
|-------------|-------------|
| フレンドリー | カジュアルで温かいトーン |
| プロフェッショナル | 丁寧でフォーマルなトーン |
| 遊び心がある | ユーモラスで明るいトーン |
| スカラー | 落ち着いた知的なトーン |

![Personality selection screen](onboarding-personality.png)

### 5. メッセンジャー連携（オプション）
TelegramやSlackなどの対応メッセンジャーとの連携を設定します。この手順はスキップして、後で設定で構成できます。

![Messenger integration screen](onboarding-messenger.png)

### 6. 完了

![Onboarding complete screen](onboarding-complete.png)
**開始**をクリックしてメイン画面に入ります。
