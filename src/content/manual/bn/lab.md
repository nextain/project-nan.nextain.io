## API Keys

![API Key ব্যবস্থাপনা](lab-keys.png)

ডেস্কটপ অ্যাপ এবং বাহ্যিক পরিষেবাগুলির জন্য API Keys পরিচালনা করুন:
- **নতুন Key তৈরি করুন**: একটি Key নাম লিখুন এবং তৈরি করুন
- **Key মুছুন**: অব্যবহৃত Keyগুলি সরান
- ডেস্কটপ অ্যাপ থেকে সংযোগ করার সময় Keyগুলি স্বয়ংক্রিয়ভাবে জারি করা হয়

## ইন্টিগ্রেশন

আপনার naia.nextain.io অ্যাকাউন্টকে বাহ্যিক মেসেঞ্জারগুলির সাথে লিঙ্ক করুন যাতে আপনি যেকোনো জায়গা থেকে AI avatar এর সাথে চ্যাট করতে পারেন।

### Discord
- Discord দিয়ে সাইন ইন করার সময় স্বয়ংক্রিয়ভাবে লিঙ্ক করা হয়
- **সেটিংস > ইন্টিগ্রেশন** এ বটের আমন্ত্রণ লিঙ্কটি খুঁজুন
- আপনার সার্ভারে @mention অথবা DM এর মাধ্যমে চ্যাট করুন
- আপনার ল্যাব অ্যাকাউন্ট থেকে ক্রেডিট কাটা হবে

### Google Chat
- Google দিয়ে সাইন ইন করার সময় উপলব্ধ
- Workspace অ্যাপ রেজিস্ট্রেশনের পর Google Chat এর মাধ্যমে চ্যাট করুন
- আপনার ল্যাব অ্যাকাউন্ট থেকে ক্রেডিট কাটা হবে
