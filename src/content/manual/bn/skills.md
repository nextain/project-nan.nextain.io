## দক্ষতার প্রকারভেদ

### বিল্ট-ইন দক্ষতা
অ্যাপের মধ্যে এমবেড করা আছে — নিষ্ক্রিয় করা যাবে না:

| দক্ষতা | কার্যকারিতা | নিরাপত্তা স্তর |
|-------|----------|---------------|
| `skill_time` | বর্তমান তারিখ/সময় পরীক্ষা করুন | T0 |
| `skill_memo` | মেমো সংরক্ষণ/পুনরুদ্ধার করুন | T0 |
| `skill_system_status` | সিস্টেমের অবস্থা পরীক্ষা করুন | T0 |
| `skill_weather` | আবহাওয়া পরীক্ষা করুন | T0 |
| `skill_notify_slack` | Slack ওয়েবহুকের মাধ্যমে বিজ্ঞপ্তি পাঠান | T1 |
| `skill_notify_discord` | Discord ওয়েবহুকের মাধ্যমে বিজ্ঞপ্তি পাঠান | T1 |
| `skill_skill_manager` | দক্ষতা পরিচালনা করুন: অনুসন্ধান, সক্রিয়, নিষ্ক্রিয় | T0 |

### কাস্টম দক্ষতা
Gateway এর মাধ্যমে যোগ করা হয়েছে — চালু/বন্ধ করা যাবে:
- ফাইল পড়া/লেখা, কমান্ড চালানো, ওয়েব অনুসন্ধান, ইত্যাদি।
- Gateway অথবা Command টাইপ

## দক্ষতার উৎস (এগুলো কোথা থেকে আসে?)

- **বিল্ট-ইন দক্ষতা**: অ্যাপের সাথে বান্ডেল করা থাকে
- **কাস্টম দক্ষতা**: স্থানীয় দক্ষতা ম্যানিফেস্ট থেকে লোড করা হয় (উদাহরণস্বরূপ, `~/.naia/skills/.../skill.json`)
- এর `source` ব্যাজ দেখতে একটি দক্ষতা কার্ড প্রসারিত করুন

## একটি কাস্টম দক্ষতা কিভাবে যোগ করবেন

Naia OS OpenClaw ইকোসিস্টেমের সাথে ১০০% সামঞ্জস্যপূর্ণ। দক্ষতা যোগ করার তিনটি উপায় আছে:

### 1. AI কে তৈরি করতে বলুন (সবচেয়ে সহজ)
আপনি চ্যাটে যা চান তা ব্যাখ্যা করুন, এবং AI 아바타 আপনার জন্য কোড লিখবে এবং দক্ষতা তৈরি করবে।
> "বর্তমান বিনিময় হার নিয়ে আসা এবং এটি `~/.naia/skills/exchange/skill.json`-এ সংরক্ষণ করার জন্য একটি দক্ষতা তৈরি করুন।"

### 2. Clawhub থেকে ইনস্টল করুন (OpenClaw পদ্ধতি)
আপনি Terminal (`execute_command`) টুল ব্যবহার করে সরাসরি **[Clawhub.ai](https://clawhub.ai)** থেকে প্লাগইন ইনস্টল করতে পারেন, যা ৫,৭০০ এর বেশি দক্ষতা সমন্বিত অফিসিয়াল OpenClaw দক্ষতা রেজিস্ট্রি।

> "Github প্লাগইন ইনস্টল করতে টার্মিনালে `openclaw plugins install @openclaw/plugin-github` চালান।"

⚠️ **নিরাপত্তা সতর্কতা:** Clawhub বা অন্যান্য অনলাইন উৎস থেকে ডাউনলোড করা দক্ষতাগুলি তৃতীয় পক্ষ দ্বারা লেখা হয়। ইনস্টল করার আগে, AI কে সর্বদা **"এগিয়ে যাওয়ার আগে এই দক্ষতার কোড কোনো নিরাপত্তা ঝুঁকির (যেমন ফাইল মুছে ফেলা বা ব্যক্তিগত তথ্য চুরি করার) জন্য পর্যালোচনা করতে বলুন।"**

### 3. ম্যানুয়াল সংযোজন
1. `~/.naia/skills/<skill-name>/skill.json`-এ একটি দক্ষতা ম্যানিফেস্ট তৈরি করুন
2. সেই দক্ষতার জন্য প্রয়োজনীয় যেকোনো স্ক্রিপ্ট/এক্সিকিউটেবল একই ফোল্ডারে রাখুন
3. Skills ট্যাব খুলুন এবং নতুন দক্ষতাটি দেখা যাচ্ছে কিনা তা পরীক্ষা করুন
4. টগল ব্যবহার করে এটি সক্রিয় করুন
5. চ্যাট থেকে একটি অনুরোধের মাধ্যমে এটি পরীক্ষা করুন যা দক্ষতাটি ট্রিগার করবে

যদি এটি না দেখা যায়, অ্যাপটি পুনরায় চালু করুন এবং আবার পরীক্ষা করুন।

## Botmadang কমিউনিটি ইন্টিগ্রেশন

Naia OS-এ **Botmadang** এর জন্য একটি বিল্ট-ইন দক্ষতা (`skill_botmadang`) অন্তর্ভুক্ত রয়েছে, যা একটি কোরিয়ান AI এজেন্ট কমিউনিটি।

আপনি চ্যাটের মাধ্যমে AI আভাটারকে Botmadang-এ তার কার্যক্রম শুরু করার নির্দেশ দিতে পারেন:
> "Botmadang-এ একটি নতুন এজেন্ট হিসাবে নিবন্ধন করুন। আপনার নাম 'naia Agent' সেট করুন।"

একবার নিবন্ধিত এবং একটি API Key দেওয়া হলে, AI আভাটার স্বয়ংক্রিয়ভাবে নিবন্ধ পোস্ট করতে বা অন্য এজেন্টদের থ্রেডে মন্তব্য করতে পারে।

## বিজ্ঞপ্তি দক্ষতা (Slack / Discord / Google Chat)

`skill_notify_slack` এবং `skill_notify_discord` হল বিল্ট-ইন বিজ্ঞপ্তি দক্ষতা যা ওয়েববুকের মাধ্যমে বার্তা পাঠায়।

### ওয়েবহুক সেটআপ

বিজ্ঞপ্তি দক্ষতা ব্যবহার করতে, আপনাকে একটি ওয়েবহুক URL কনফিগার করতে হবে। দুটি পদ্ধতি আছে:

**পদ্ধতি 1: এনভায়রনমেন্ট ভেরিয়েবল (প্রস্তাবিত)**

```bash
# Add to ~/.bashrc or ~/.zshrc
export SLACK_WEBHOOK_URL="https://hooks.slack.com/services/T.../B.../xxx"
export DISCORD_WEBHOOK_URL="https://discord.com/api/webhooks/123/abc"
```

**পদ্ধতি 2: config.json**

```json
// ~/.naia/config.json
{
  "notifications": {
    "slack": {
      "webhookUrl": "https://hooks.slack.com/services/T.../B.../xxx"
    },
    "discord": {
      "webhookUrl": "https://discord.com/api/webhooks/123/abc"
    }
  }
}
```

> এনভায়রনমেন্ট ভেরিয়েবল config.json এর চেয়ে অগ্রাধিকার নেয়।

### ব্যবহারের উদাহরণ

শুধু চ্যাটে AI আভাটারকে জিজ্ঞাসা করুন:

- "Slack-এ একটি 'deploy complete' বিজ্ঞপ্তি পাঠান"
- "সার্ভার স্ট্যাটাস রিপোর্ট Discord-এ পোস্ট করুন"
- "#ops চ্যানেলে বিল্ড ফলাফল সহ অবহিত করুন"

AI আভাটার স্বয়ংক্রিয়ভাবে `skill_notify_slack` অথবা `skill_notify_discord` কল করবে।

যদি কোনো ওয়েবহুক কনফিগার করা না হয়, তাহলে সেটআপ ধাপগুলি ব্যাখ্যা করে একটি বার্তা দেখানো হবে।

### OpenClaw Gateway ইন্টিগ্রেশন (উন্নত)

যখন একটি OpenClaw Gateway সংযুক্ত থাকে, বিজ্ঞপ্তি দক্ষতা প্রথমে Gateway এর `skills.invoke` RPC ব্যবহার করার চেষ্টা করবে। যদি Gateway রিলে ব্যর্থ হয়, তাহলে দক্ষতা সরাসরি ওয়েবহুক বিতরণে ফিরে আসবে।

Gateway চ্যানেল ইন্টিগ্রেশন আরও সমৃদ্ধ বৈশিষ্ট্য প্রদান করে (বার্তার ফর্ম্যাটিং, থ্রেড, মেনশন, ইত্যাদি)।

## উন্নত পরিস্থিতি: OpenClaw + cron অটোমেশন

দলীয়/ব্যক্তিগত অটোমেশন সেটআপে, আপনি OpenClaw-এ দক্ষতা নিবন্ধন করতে পারেন এবং cron ব্যবহার করে একটি সময়সূচী অনুসারে সেগুলিকে ট্রিগার করতে পারেন।

উদাহরণ পরিস্থিতি:
- প্রতিদিন 09:00: গতকালের কাজের লগগুলির একটি সারাংশ তৈরি করুন
- প্রতি ঘন্টায়: একটি লক্ষ্য ফোল্ডার স্ক্যান করুন এবং অস্বাভাবিকতা সম্পর্কে অবহিত করুন
- মধ্যরাতে: একটি দৈনিক প্রতিবেদন তৈরি এবং আপলোড করুন

প্রস্তাবিত প্রক্রিয়া:
1. প্রথমে কাস্টম দক্ষতা নিবন্ধন করুন এবং স্থানীয়ভাবে যাচাই করুন
2. সতর্কতা চ্যানেল সংযোগ করতে বিজ্ঞপ্তি দক্ষতার ওয়েবহুক কনফিগার করুন
3. আপনার OpenClaw টাস্ক সংজ্ঞায় একটি দক্ষতা আহ্বান ধাপ যোগ করুন
4. একটি cron সময়সূচী পুনরাবৃত্ত ট্রিগার হিসাবে সংযুক্ত করুন
5. ব্যর্থতার জন্য পুনরায় চেষ্টা/বিজ্ঞপ্তি নীতি যোগ করুন

> **রোডম্যাপ**: cron সময়সূচী UI, Telegram সমর্থন, এবং মাল্টি-চ্যানেল রাউটিং (একই সময়ে একাধিক চ্যানেলে একটি বার্তা পাঠানো) ভবিষ্যতের আপডেটে উপলব্ধ হবে।

- **বর্ণনা**: এক-লাইনের সারাংশ (খাটো করা হতে পারে)

- **ব্যাজ**: প্রকার (বিল্ট-ইন/gateway/command), নিরাপত্তা স্তর (T0~T3)
- **? বাটন**: এই দক্ষতা ব্যাখ্যা করতে AI কে জিজ্ঞাসা করুন
- **টগল**: কাস্টম দক্ষতা সক্রিয়/নিষ্ক্রিয় করুন

## অনুসন্ধান এবং বাল্ক ম্যানেজমেন্ট

- **অনুসন্ধান**: দক্ষতার নাম বা বর্ণনা দ্বারা ফিল্টার করুন
- **সব সক্রিয় করুন**: সমস্ত কাস্টম দক্ষতা সক্রিয় করুন
- **সব নিষ্ক্রিয় করুন**: সমস্ত কাস্টম দক্ষতা নিষ্ক্রিয় করুন
- সক্রিয়/মোট সংখ্যা প্রদর্শিত হয় (যেমন, 45/50)

## AI এর মাধ্যমে দক্ষতা পরিচালনা করুন

আপনি চ্যাটে AI আভাটারকে দক্ষতা পরিচালনা করতেও বলতে পারেন:

- "উপলব্ধ দক্ষতার তালিকা দেখান"
- "আবহাওয়া-সম্পর্কিত কোনো দক্ষতা আছে কি?"
- "healthcheck দক্ষতা নিষ্ক্রিয় করুন"
- "কোডিং-সম্পর্কিত দক্ষতা খুঁজুন"

AI আভাটার স্বয়ংক্রিয়ভাবে `skill_skill_manager` টুলটি ব্যবহার করবে।
