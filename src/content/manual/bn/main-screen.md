## অবতার এলাকা (উপরে)

- আপনার নির্বাচিত VRM চরিত্র 3D-তে রেন্ডার হয়
- কথোপকথনের উপর ভিত্তি করে **আবেগ** পরিবর্তিত হয়:
  - [HAPPY] — আনন্দ
  - [SAD] — দুঃখ
  - [ANGRY] — রাগ
  - [SURPRISED] — বিস্ময়
  - [NEUTRAL] — নিরপেক্ষ
  - [THINK] — চিন্তা
- **ক্যামেরা নিয়ন্ত্রণ**: দৃশ্যের স্থান পরিবর্তন করতে অবতার এলাকায় টেনে ধরুন
- **পটভূমি চিত্র**: অবতারের পিছনে প্রদর্শিত হয়
- **লিপ-সিঙ্ক**: TTS প্রতিক্রিয়ার সময় অ্যানিমেশন চলে

## প্যানেল এলাকা (নিচে)

নিচের প্যানেলে ৮টি ট্যাব রয়েছে:

| ট্যাব | বিবরণ |
|-----|-------------|
| চ্যাট | Conversation with AI 아바타 |
| ইতিহাস | পূর্ববর্তী কথোপকথন |
| অগ্রগতি | টুল এক্সিকিউশন/ত্রুটি পর্যবেক্ষণ |
| দক্ষতা | উপলব্ধ দক্ষতা (টুলস) তালিকা |
| চ্যানেল | সংযুক্ত মেসেঞ্জার চ্যানেলগুলি পরিচালনা করুন |
| এজেন্ট | ব্যাকগ্রাউন্ড এজেন্ট এবং সেশন ব্যবস্থাপনা |
| ডায়াগনস্টিকস | গেটওয়ে সিস্টেমের স্বাস্থ্য এবং লগস |
| সেটিংস | সমস্ত অ্যাপ সেটিংস |

বিস্তারিত বৈশিষ্ট্যের জন্য প্রতিটি ট্যাবের পৃষ্ঠা দেখুন।
