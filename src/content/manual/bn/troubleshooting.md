## এআই এজেন্ট সাড়া দিচ্ছে না

১. Settings > AI Settings-এ API key বা Lab সংযোগ পরীক্ষা করুন
২. ইন্টারনেট সংযোগ যাচাই করুন
৩. Settings > Lab Account-এ ক্রেডিট ব্যালেন্স পরীক্ষা করুন
৪. নিশ্চিত করুন যে মডেলটি সঠিকভাবে কনফিগার করা হয়েছে

## ভয়েস কাজ করছে না

১. Settings > Voice-এ TTS/STT সক্রিয় আছে কিনা নিশ্চিত করুন
২. একটি Google API key প্রয়োজন হতে পারে
৩. মাইক্রোফোন অনুমতি (OS সেটিংস) পরীক্ষা করুন
৪. ব্রাউজার মিডিয়া অনুমতিগুলি ব্লক করা হয়নি তা নিশ্চিত করুন

## ক্রেডিট কম

১. Settings > Lab Account-এ ব্যালেন্স পরীক্ষা করুন
২. Lab Dashboard-এ ব্যবহার পর্যালোচনা করুন
৩. বিনামূল্যে ক্রেডিট মাসিক রিচার্জ হয়
৪. ১০০ ক্রেডিটের জন্য BASIC ($10/mo)-এ আপগ্রেড করুন

## টুলস কাজ করছে না

১. Settings > Tools-এ "Enable Tools" চালু আছে কিনা নিশ্চিত করুন
২. Gateway চলছে কিনা পরীক্ষা করুন
৩. Skills tab-এ প্রয়োজনীয় দক্ষতাগুলি সক্রিয় আছে কিনা যাচাই করুন
৪. সাধারণ ব্যবহারকারী পরিবেশে, আপনাকে ম্যানুয়ালি Gateway URL/token প্রবেশ করতে হবে না
৫. শুধুমাত্র সেলফ-হোস্টেড/উন্নত সেটআপের জন্য Gateway URL/token পরীক্ষা করুন

## অ্যাপ ধীর বা জমে গেছে

১. একটি নতুন কথোপকথন শুরু করুন (+) — দীর্ঘ চ্যাট গতি কমিয়ে দিতে পারে
২. অ্যাপটি রিস্টার্ট করুন
৩. সম্পূর্ণ রিসেটের জন্য Settings > Reset All ব্যবহার করুন

## আবহাওয়া দেখাচ্ছে না

- আবহাওয়া দক্ষতার জন্য একটি ইন্টারনেট সংযোগ প্রয়োজন
- সুনির্দিষ্ট হন: "Tell me the weather in Seoul"
- ইংরেজি শহরের নাম দিয়ে চেষ্টা করুন (যেমন, "Seoul weather")

## ল্যাব লগইন কাজ করছে না

১. ইন্টারনেট সংযোগ পরীক্ষা করুন
২. আপনার ব্রাউজারে সরাসরি naia.nextain.io অ্যাক্সেস করার চেষ্টা করুন
৩. আপনি আপনার Google অ্যাকাউন্ট দিয়ে সাইন ইন করতে পারছেন কিনা যাচাই করুন
৪. অ্যাপটি রিস্টার্ট করুন এবং আবার চেষ্টা করুন
