## Résumé des statistiques

Quatre statistiques en haut :

| Élément | Description |
|---|---|
| **Total des événements** | Nombre total d'opérations |
| **Coût total** | Coûts d'API utilisés |
| **Outils utilisés** | Nombre d'outils invoqués |
| **Erreurs** | Nombre d'erreurs rencontrées |

## Liste des événements

- Chaque événement affiche le type, l'horodatage et les détails
- Bouton **Actualiser** pour les dernières données
- Filtrer pour afficher uniquement les erreurs
