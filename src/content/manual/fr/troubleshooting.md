Problèmes courants et solutions.

## L'agent IA ne répond pas

1. Vérifiez la clé API ou la connexion au Lab dans Paramètres > Paramètres IA
2. Vérifiez la connexion internet
3. Vérifiez le solde de crédits dans Paramètres > Compte Lab
4. Assurez-vous que le modèle est correctement configuré

## La voix ne fonctionne pas

1. Assurez-vous que TTS/STT est activé dans Paramètres > Voix
2. Une clé API Google peut être requise
3. Vérifiez les permissions du microphone (paramètres du système d'exploitation)
4. Assurez-vous que les permissions médias du navigateur ne sont pas bloquées

## Faible niveau de crédits

1. Vérifiez le solde dans Paramètres > Compte Lab
2. Examinez l'utilisation dans le Tableau de bord du Lab
3. Les crédits gratuits sont rechargés mensuellement
4. Passez à BASIC (10 $/mois) pour 100 crédits

## Les outils ne fonctionnent pas

1. Assurez-vous que "Activer les outils" est activé dans Paramètres > Outils
2. Vérifiez que Gateway est en cours d'exécution
3. Vérifiez que les compétences nécessaires sont activées dans l'onglet Compétences
4. Dans les environnements utilisateur normaux, vous n'avez pas besoin de saisir manuellement l'URL/le jeton de Gateway
5. Vérifiez l'URL/le jeton de Gateway uniquement pour les configurations auto-hébergées/avancées

## L'application est lente ou bloquée

1. Démarrez une nouvelle conversation (+) — les longues discussions peuvent ralentir l'application
2. Redémarrez l'application
3. Utilisez Paramètres > Réinitialiser tout pour une réinitialisation complète

## La météo ne s'affiche pas

- La compétence météo nécessite une connexion internet
- Soyez précis : "Donne-moi la météo à Séoul"
- Essayez avec des noms de villes en anglais (par exemple, "Seoul weather")

## La connexion au Lab ne fonctionne pas

1. Vérifiez la connexion internet
2. Essayez d'accéder directement à naia.nextain.io dans votre navigateur
3. Vérifiez que vous pouvez vous connecter avec votre compte Google
4. Redémarrez l'application et réessayez
