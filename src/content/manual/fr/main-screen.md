L'écran principal comporte deux zones.

![Disposition de l'écran principal](main-screen.png)

## Zone de l'avatar (Haut)

- Votre personnage VRM choisi est rendu en 3D
- Les **émotions** changent en fonction de la conversation :
  - [HAPPY] — Joie
  - [SAD] — Tristesse
  - [ANGRY] — Colère
  - [SURPRISED] — Surprise
  - [NEUTRAL] — Neutre
  - [THINK] — Réflexion
- **Contrôle de la caméra** : Faites glisser dans la zone de l'avatar pour déplacer le point de vue
- **Image d'arrière-plan** : Apparaît derrière l'avatar
- **Synchronisation labiale** : L'animation est jouée pendant les réponses TTS

## Zone du panneau (Bas)

Le panneau inférieur comporte 8 onglets :

| Onglet | Description |
|---|---|
| Chat | Conversation avec l'IA 아바타 |
| Historique | Conversations précédentes |
| Progression | Suivi de l'exécution des outils/des erreurs |
| Compétences | Liste des compétences (outils) disponibles |
| Canaux | Gérer les canaux de messagerie connectés |
| Agents | Agents en arrière-plan et gestion des sessions |
| Diagnostics | État et journaux du système de passerelle |
| Paramètres | Tous les paramètres de l'application |

Consultez la page de chaque onglet pour connaître les fonctionnalités détaillées.
