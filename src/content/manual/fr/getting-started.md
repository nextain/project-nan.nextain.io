Lorsque vous lancez l'application pour la première fois, un assistant d'intégration apparaît. Il y a deux façons de commencer.

## Commencer avec Naia OS (Recommandé)

La façon la plus simple — gratuite et sans API Key nécessaire.

![Écran d'intégration — sélection du fournisseur](onboarding-provider.png)

1. Cliquez sur le bouton **Naia OS** sur l'écran d'intégration
2. Un navigateur s'ouvre sur la page de connexion naia.nextain.io
3. Connectez-vous avec votre **compte Google** (l'inscription est automatique)
4. Vous serez automatiquement redirigé vers l'application de bureau
5. Vous recevez **20 crédits gratuits** lors de l'inscription

## Commencer avec l'API Key

Vous pouvez également saisir une API Key de fournisseurs comme Google Gemini.

![Écran d'intégration — saisie de l'API Key](onboarding-apikey.png)

1. Sélectionnez un fournisseur (Gemini, etc.)
2. Cliquez sur **Next**
3. Saisissez votre API Key
4. Cliquez sur **Validate** pour tester la clé

## Étapes d'intégration

Après la connexion au Lab ou la saisie de l'API Key, suivez ces étapes :

### 1. Nommez votre IA
Donnez à AI 아바타 le nom que vous souhaitez (par exemple, "AI 아바타", "Luna", "Cookie").

![Saisie du nom de l'IA](onboarding-agent-name.png)

### 2. Votre Nom
Saisissez comment AI 아바타 doit vous appeler.

![Saisie de votre nom](onboarding-user-name.png)

### 3. Choisissez un personnage
Choisissez parmi 4 avatars VRM :

| Personnage | Description |
|-----------|-------------|
| Shino (Sombre) | Uniforme sombre |
| Shino (Clair) | Uniforme clair |
| Fille | Fille générique |
| Garçon | Garçon générique |

![Écran de sélection de personnage](onboarding-character.png)

### 4. Choisissez une personnalité
Choisissez parmi 4 préréglages :

| Personnalité | Description |
|-------------|-------------|
| Amicale | Ton décontracté et chaleureux |
| Professionnelle | Ton poli et formel |
| Ludique | Ton humoristique et joyeux |
| Érudite | Ton calme et intellectuel |

![Écran de sélection de personnalité](onboarding-personality.png)

### 5. Intégration de messagerie (Optionnel)
Configurez l'intégration avec les messageries prises en charge comme Telegram ou Slack. Cette étape peut être ignorée et configurée plus tard dans les paramètres.

![Écran d'intégration de messagerie](onboarding-messenger.png)

### 6. Terminé

![Écran d'intégration terminée](onboarding-complete.png)
Cliquez sur **Get Started** pour accéder à l'écran principal.
