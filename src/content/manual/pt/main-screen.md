A tela principal tem duas áreas.

![Layout da tela principal](main-screen.png)

## Área do Avatar (Superior)

- Seu personagem VRM escolhido é renderizado em 3D
- As **Emoções** mudam com base na conversa:
  - [HAPPY] — Alegria
  - [SAD] — Tristeza
  - [ANGRY] — Raiva
  - [SURPRISED] — Surpresa
  - [NEUTRAL] — Neutro
  - [THINK] — Pensando
- **Controle da câmera**: Arraste na área do avatar para mover o ponto de vista
- **Imagem de fundo**: Aparece atrás do avatar
- **Sincronização labial**: Animação é reproduzida durante as respostas de TTS

## Área do Painel (Inferior)

O painel inferior tem 8 abas:

| Aba | Descrição |
|-----|-------------|
| Chat | Conversa com AI 아바타 |
| Histórico | Conversas anteriores |
| Progresso | Monitoramento de execução/erros de ferramentas |
| Habilidades | Lista de habilidades (ferramentas) disponíveis |
| Canais | Gerenciar canais de mensagens conectados |
| Agentes | Agentes em segundo plano e gerenciamento de sessão |
| Diagnósticos | Saúde do sistema de gateway e logs |
| Configurações | Todas as configurações do aplicativo |

Consulte a página de cada aba para recursos detalhados.
