## Resumo das Estatísticas

Quatro estatísticas no topo:

| Item | Descrição |
|------|-------------|
| **Total de Eventos** | Número total de operações |
| **Custo Total** | Custos de API utilizados |
| **Ferramentas Usadas** | Número de ferramentas invocadas |
| **Erros** | Número de erros encontrados |

## Lista de Eventos

- Cada evento mostra tipo, carimbo de data/hora e detalhes
- Botão **Atualizar** para os dados mais recentes
- Filtrar para mostrar apenas erros
