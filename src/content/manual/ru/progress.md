## Сводка статистики

Четыре показателя вверху:

| Пункт | Описание |
|------|-------------|
| **Всего событий** | Общее количество операций |
| **Общая стоимость** | Использованные расходы API |
| **Использовано инструментов** | Количество вызванных инструментов |
| **Ошибки** | Количество обнаруженных ошибок |

## Список событий

- Каждое событие показывает тип, метку времени и подробности
- Кнопка **Обновить** для получения последних данных
- Фильтр для отображения только ошибок
