Общие проблемы и решения.

## Агент ИИ не отвечает

1. Проверьте API key или подключение к Lab в **Settings > AI Settings**
2. Проверьте подключение к интернету
3. Проверьте баланс кредитов в **Settings > Lab Account**
4. Убедитесь, что модель настроена правильно

## Голосовой ввод/вывод не работает

1. Убедитесь, что TTS/STT включен в **Settings > Voice**
2. Может потребоваться Google API key
3. Проверьте разрешения микрофона (настройки ОС)
4. Убедитесь, что разрешения медиа для браузера не заблокированы

## Мало кредитов

1. Проверьте баланс в **Settings > Lab Account**
2. Просмотрите использование в Lab Dashboard
3. Бесплатные кредиты пополняются ежемесячно
4. Обновитесь до BASIC (10 $/мес) для получения 100 кредитов

## Инструменты не работают

1. Убедитесь, что "Enable Tools" включен в **Settings > Tools**
2. Проверьте, что Gateway запущен
3. Убедитесь, что необходимые навыки включены во вкладке Skills
4. В обычных пользовательских средах вам не нужно вводить Gateway URL/token вручную
5. Проверяйте Gateway URL/token только для самостоятельно размещенных/расширенных настроек

## Приложение работает медленно или зависло

1. Начните новую беседу (+) — длинные чаты могут замедлять работу
2. Перезапустите приложение
3. Используйте **Settings > Reset All** для полного сброса

## Погода не отображается

- Навык погоды требует подключения к интернету
- Будьте конкретны: "Расскажи мне погоду в Сеуле"
- Попробуйте использовать названия городов на английском языке (например, "Seoul weather")

## Вход в Lab не работает

1. Проверьте подключение к интернету
2. Попробуйте получить доступ к naia.nextain.io напрямую в браузере
3. Убедитесь, что вы можете войти с помощью своей учетной записи Google
4. Перезапустите приложение и попробуйте снова
