## Учетная запись Naia OS

Управление интеграцией с Lab.

### При подключении

- Отображается ID пользователя
- Отображается **баланс кредитов** (запрос в реальном времени)
- Кнопка **Панель управления**: Открывает панель управления Lab в браузере
- Кнопка **Пополнить кредиты**: Переход на страницу оплаты Lab
- Кнопка **Отключиться**: Отвязывает учетную запись Lab
- Настройки автоматически синхронизируются с Lab при сохранении

### При отключении
- Кнопка **Вход в Lab**: Вход через браузер, автоматическое подключение

## Настройки ИИ

- **Провайдер**: Gemini (по умолчанию), OpenAI, Claude, Grok, zAI, Ollama
  - В настоящее время доступен только Gemini; остальные скоро появятся

- **API Key**: API-ключ провайдера (не требуется при использовании Lab)

## Голос (TTS/STT)

- **Голосовой отклик (TTS)**: ВКЛ/ВЫКЛ
- **Голосовой ввод (STT)**: ВКЛ/ВЫКЛ
- **Google API Key**: Выделенный ключ TTS/STT (оставьте пустым для использования ключа чата)
- **Голос TTS**: Выберите один из 11 корейских голосов
  - Neural2: Высокое качество (16 $/1M символов)
  - WaveNet: Естественный (16 $/1M символов)
  - Standard: Базовый (4 $/1M символов)
- Кнопка **Предпросмотр** для тестирования выбранного голоса

## Инструменты

Управление разрешениями системного контроля, доступными для AI avatar.

- **Включить инструменты**: Общее ВКЛ/ВЫКЛ для функций инструментов
- **Gateway URL / Gateway Token**: Автоматически управляются в обычных пользовательских средах, поэтому ручной ввод не требуется
  - Только продвинутым пользователям, использующим самостоятельно размещенный Gateway, необходимо проверять/редактировать эти значения
- **Разрешенные инструменты**: Инструменты, помеченные как "Всегда разрешать"
  - **Очистить разрешенные инструменты**: Сбросить все одобрения

> **Руководство по уровням безопасности (T0~T3)**
> Применяется 4-уровневая система безопасности, основанная на системном воздействии при использовании инструмента ИИ.
> - **T0 (Только чтение)**: Задачи запроса. Автоматически выполняются без вмешательства пользователя.
> - **T1 (Уведомление)**: Простой внешний поиск/уведомление. Отображает предупреждение на экране при выполнении.
> - **T2 (Осторожно)**: Системные изменения, такие как создание/редактирование файлов. Требует явного одобрения пользователя.
> - **T3 (Опасно)**: Опасные задачи, такие как выполнение команд терминала. Требует одобрения каждый раз.

## Интеграции

Управляйте интеграциями с внешними мессенджерами на странице **Настройки > Интеграции**.

- **Интеграция с Discord**: Автоматически связывается при входе через Discord. Добавьте бота на свой сервер по ссылке-приглашению, чтобы общаться с ИИ через упоминания или личные сообщения.
- **Интеграция с Google Chat**: Общайтесь с ИИ через Google Chat при входе через Google.
- Статус подключения (подключено/не подключено) отображается в виде значков.

## Устройство и Кодовое слово

![Device Settings](settings-device.png)

- **Список кодовых слов**: Управление словами для активации ИИ (напр., "Hey Naia", "Hello Naia")
- **Сопряжение устройств**: Управление интеграциями внешних устройств (напр., сопряженные узлы OpenClaw) и подключениями к устройствам умного дома.
- Включает функции для одобрения устройств и сброса токенов.

## Память (Факты)

Управление фактами, которые AI 아바타 узнал из разговоров.
- Каждый факт показывает **ключ** и **значение**
- Кнопка **Удалить** для удаления ненужных фактов
- Сохраненные факты влияют на персонализацию ответов в последующих чатах
- Удаление большого количества фактов может снизить качество персонализации

## Сохранить и Сбросить

- **Сохранить**: Применить все изменения
- **Сбросить все**: Восстановить все настройки и камеру до значений по умолчанию (требуется подтверждение)
