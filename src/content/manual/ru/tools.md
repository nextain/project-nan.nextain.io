Когда инструменты включены, AI 아바타 может выполнять различные задачи напрямую.

## Доступные Инструменты

| Инструмент | Функция | Безопасность |
|------|----------|----------|
| execute_command | Запуск команд терминала | T3 |
| read_file | Чтение содержимого файлов | T0 |
| write_file | Создание/изменение файлов | T2 |
| search_files | Поиск в файловой системе | T0 |
| web_search | Поиск в интернете | T1 |
| apply_diff | Редактирование файлов (применение diff) | T2 |
| browser | Открытие веб-страниц | T1 |
| sessions_spawn | Создание субагентов | T2 |

## Примеры Использования

Вы можете использовать инструменты с помощью естественного языка:

| Запрос | Используемый Инструмент |
|---------|-----------|
| "Показать файлы в этой папке" | search_files |
| "Прочитать README.md" | read_file |
| "Запустить git status" | execute_command |
| "Найти TypeScript" | web_search |
| "Проверить этот код" | sessions_spawn |

## Подтверждение Безопасности

Каждый инструмент требует подтверждения в соответствии с его уровнем безопасности:

- **T0 (Только для чтения)**: Автоматически подтверждено — без побочных эффектов
- **T1 (Уведомление)**: Показано уведомление — внешний доступ, но без изменений
- **T2 (Осторожно)**: Требуется подтверждение пользователя — изменения файлов, создание агентов
- **T3 (Опасно)**: Требуется подтверждение пользователя — выполнение команд

При первом подтверждении выбор "Всегда разрешать" автоматически подтверждает этот инструмент для дальнейшего использования. Вы можете сбросить разрешенные инструменты в Настройках > Инструменты.

## Подключение к шлюзу

Для инструментов требуется подключение к шлюзу:

1. Включите **Включить Инструменты** в Настройках > Инструменты
2. В обычных пользовательских средах данные для подключения к шлюзу обрабатываются автоматически
3. Если инструменты не отвечают, перезапустите приложение и повторите попытку
4. Убедитесь, что необходимые навыки включены на вкладке Навыки
5. Только для самостоятельно размещенных/расширенных настроек требуется ручная проверка URL-адреса/токена шлюза
