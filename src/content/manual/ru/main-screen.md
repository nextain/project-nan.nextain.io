## Область Аватара (Верхняя)

- Выбранный вами VRM-персонаж отображается в 3D
- **Эмоции** меняются в зависимости от разговора:
  - [HAPPY] — Радость
  - [SAD] — Грусть
  - [ANGRY] — Злость
  - [SURPRISED] — Удивление
  - [NEUTRAL] — Нейтральность
  - [THINK] — Размышление
- **Управление камерой**: Перетащите в области аватара, чтобы переместить точку обзора
- **Фоновое изображение**: Появляется за аватаром
- **Синхронизация губ**: Анимация проигрывается во время ответов TTS

## Область Панели (Нижняя)

На нижней панели 8 вкладок:

| Вкладка | Описание |
|---------|----------|
| Chat | Разговор с AI аватаром |
| History | Предыдущие разговоры |
| Progress | Мониторинг выполнения инструментов/ошибок |
| Skills | Список доступных навыков (инструментов) |
| Channels | Управление подключенными каналами мессенджеров |
| Agents | Фоновые агенты и управление сессиями |
| Diagnostics | Состояние системы шлюза и логи |
| Settings | Все настройки приложения |

Подробные функции смотрите на странице каждой вкладки.
