## Интеграции

Свяжите свою учетную запись naia.nextain.io с внешними мессенджерами, чтобы общаться с ИИ-аватаром из любого места.

### Discord
- Автоматически связывается при входе через Discord
- Найдите ссылку для приглашения бота в разделе **Настройки > Интеграции**
- Общайтесь через @упоминание или личное сообщение на вашем сервере
- Кредиты списываются с вашего аккаунта Lab

### Google Chat
- Доступно при входе через Google
- Общайтесь через Google Chat после регистрации приложения Workspace
- Кредиты списываются с вашего аккаунта Lab
