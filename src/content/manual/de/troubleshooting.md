Häufige Probleme und Lösungen.

## KI-Agent antwortet nicht

1. Überprüfen Sie den API key oder die Lab-Verbindung unter Einstellungen > KI-Einstellungen
2. Überprüfen Sie die Internetverbindung
3. Überprüfen Sie das Guthaben unter Einstellungen > Lab-Konto
4. Stellen Sie sicher, dass das Modell korrekt konfiguriert ist

## Sprachausgabe funktioniert nicht

1. Stellen Sie sicher, dass TTS/STT unter Einstellungen > Sprache aktiviert ist
2. Ein Google API key könnte erforderlich sein
3. Überprüfen Sie die Mikrofonberechtigungen (Betriebssystemeinstellungen)
4. Stellen Sie sicher, dass die Medienberechtigungen des Browsers nicht blockiert sind

## Guthaben ist niedrig

1. Überprüfen Sie das Guthaben unter Einstellungen > Lab-Konto
2. Überprüfen Sie die Nutzung im Lab Dashboard
3. Kostenlose Credits werden monatlich aufgeladen
4. Führen Sie ein Upgrade auf BASIC (10 $/Monat) für 100 Credits durch

## Tools funktionieren nicht

1. Stellen Sie sicher, dass "Tools aktivieren" unter Einstellungen > Tools eingeschaltet ist
2. Überprüfen Sie, ob Gateway läuft
3. Überprüfen Sie, ob die benötigten Skills im Tab "Skills" aktiviert sind
4. In normalen Benutzerumgebungen müssen Sie die Gateway URL/den Gateway token nicht manuell eingeben
5. Überprüfen Sie die Gateway URL/den Gateway token nur für selbstgehostete/erweiterte Setups

## App ist langsam oder eingefroren

1. Starten Sie eine neue Konversation (+) — lange Chats können die Geschwindigkeit verringern
2. Starten Sie die App neu
3. Verwenden Sie Einstellungen > Alles zurücksetzen für einen vollständigen Reset

## Wetter wird nicht angezeigt

- Der Wetter-Skill erfordert eine Internetverbindung
- Seien Sie spezifisch: "Sagen Sie mir das Wetter in Seoul"
- Versuchen Sie es mit englischen Stadtnamen (z.B. "Seoul weather")

## Lab-Login funktioniert nicht

1. Überprüfen Sie die Internetverbindung
2. Versuchen Sie, naia.nextain.io direkt in Ihrem Browser aufzurufen
3. Vergewissern Sie sich, dass Sie sich mit Ihrem Google-Konto anmelden können
4. Starten Sie die App neu und versuchen Sie es erneut
