Der Hauptbildschirm besteht aus zwei Bereichen.

![Layout des Hauptbildschirms](main-screen.png)

## Avatar-Bereich (Oben)

- Ihr ausgewählter VRM-Charakter wird in 3D gerendert
- **Emotionen** ändern sich basierend auf der Konversation:
  - [HAPPY] – Freude
  - [SAD] – Traurigkeit
  - [ANGRY] – Wut
  - [SURPRISED] – Überraschung
  - [NEUTRAL] – Neutral
  - [THINK] – Nachdenken
- **Kamerasteuerung**: Ziehen Sie im Avatar-Bereich, um den Blickpunkt zu verschieben
- **Hintergrundbild**: Erscheint hinter dem Avatar
- **Lip-sync**: Animation wird während TTS-Antworten abgespielt

## Panel-Bereich (Unten)

Das untere Panel hat 8 Registerkarten:

| Registerkarte | Beschreibung |
|---------------|--------------|
| Chat | Unterhaltung mit AI 아바타 |
| Verlauf | Vorherige Unterhaltungen |
| Fortschritt | Überwachung der Tool-Ausführung/Fehler |
| Fähigkeiten | Liste der verfügbaren Fähigkeiten (Tools) |
| Kanäle | Verbundene Messenger-Kanäle verwalten |
| Agenten | Hintergrundagenten und Sitzungsverwaltung |
| Diagnostik | Gateway-Systemzustand und Protokolle |
| Einstellungen | Alle App-Einstellungen |

Detaillierte Funktionen finden Sie auf der Seite jeder Registerkarte.
