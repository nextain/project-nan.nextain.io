Layar utama memiliki dua area.

![Tata letak layar utama](main-screen.png)

## Area Avatar (Atas)

- Karakter VRM pilihan Anda dirender dalam 3D
- **Emosi** berubah berdasarkan percakapan:
  - [HAPPY] — Kegembiraan
  - [SAD] — Kesedihan
  - [ANGRY] — Kemarahan
  - [SURPRISED] — Kejutan
  - [NEUTRAL] — Netral
  - [THINK] — Berpikir
- **Kontrol kamera**: Seret di area avatar untuk memindahkan sudut pandang
- **Gambar latar belakang**: Muncul di belakang avatar
- **Lip-sync**: Animasi diputar selama respons TTS

## Area Panel (Bawah)

Panel bawah memiliki 8 tab:

| Tab | Deskripsi |
|-----|-------------|
| Chat | Percakapan dengan AI 아바타 |
| Riwayat | Percakapan sebelumnya |
| Progres | Pemantauan eksekusi/kesalahan alat |
| Keterampilan | Daftar keterampilan (alat) yang tersedia |
| Saluran | Kelola saluran messenger yang terhubung |
| Agen | Agen latar belakang dan manajemen sesi |
| Diagnostik | Kesehatan sistem gateway dan log |
| Pengaturan | Semua pengaturan aplikasi |

Lihat halaman setiap tab untuk fitur-fitur yang lebih detail.
