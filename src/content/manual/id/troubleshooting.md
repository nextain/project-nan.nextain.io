## Agen AI tidak merespons

1. Periksa API key atau koneksi Lab di Pengaturan > Pengaturan AI
2. Verifikasi koneksi internet
3. Periksa saldo kredit di Pengaturan > Akun Lab
4. Pastikan model dikonfigurasi dengan benar

## Suara tidak berfungsi

1. Pastikan TTS/STT diaktifkan di Pengaturan > Suara
2. Mungkin diperlukan Google API key
3. Periksa izin mikrofon (pengaturan OS)
4. Pastikan izin media browser tidak diblokir

## Kredit rendah

1. Periksa saldo di Pengaturan > Akun Lab
2. Tinjau penggunaan di Dasbor Lab
3. Kredit gratis diisi ulang setiap bulan
4. Tingkatkan ke BASIC ($10/bulan) untuk 100 kredit

## Alat tidak berfungsi

1. Pastikan "Aktifkan Alat" menyala di Pengaturan > Alat
2. Periksa apakah Gateway sedang berjalan
3. Verifikasi keterampilan yang diperlukan di tab Keterampilan
4. Di lingkungan pengguna normal, Anda tidak perlu memasukkan URL/token Gateway secara manual
5. Periksa URL/token Gateway hanya untuk pengaturan mandiri/lanjutan

## Cuaca tidak ditampilkan

- Keterampilan cuaca memerlukan koneksi internet
- Bersikaplah spesifik: "Beri tahu saya cuaca di Seoul"
- Coba dengan nama kota berbahasa Inggris (misalnya, "cuaca Seoul")

## Login Lab tidak berfungsi

1. Periksa koneksi internet
2. Coba akses naia.nextain.io langsung di browser Anda
3. Verifikasi Anda dapat masuk dengan akun Google Anda
4. Mulai ulang aplikasi dan coba lagi
