## Ringkasan Statistik

Empat statistik di bagian atas:

| Item | Deskripsi |
|------|-------------|
| **Total Kejadian** | Jumlah total operasi |
| **Total Biaya** | Biaya API yang digunakan |
| **Alat Digunakan** | Jumlah alat yang dipanggil |
| **Kesalahan** | Jumlah kesalahan yang ditemukan |

## Daftar Kejadian

- Setiap kejadian menunjukkan jenis, stempel waktu, dan detail
- Tombol **Segarkan** untuk data terbaru
- Filter untuk menampilkan kesalahan saja
