## منطقة الأفاتار (العلوية)

- يتم عرض شخصية VRM التي اخترتها ثلاثية الأبعاد
- تتغير **المشاعر** بناءً على المحادثة:
  - [HAPPY] — بهجة
  - [SAD] — حزن
  - [ANGRY] — غضب
  - [SURPRISED] — دهشة
  - [NEUTRAL] — محايد
  - [THINK] — تفكير
- **التحكم بالكاميرا**: اسحب في منطقة الأفاتار لتحريك نقطة العرض
- **صورة الخلفية**: تظهر خلف الأفاتار
- **مزامنة الشفاه**: يتم تشغيل الرسوم المتحركة أثناء استجابات TTS

## منطقة اللوحة (السفلية)

تحتوي اللوحة السفلية على 8 علامات تبويب:

| علامة التبويب | الوصف |
|---------------|-------|
| الدردشة | محادثة مع AI 아바타 |
| السجل | المحادثات السابقة |
| التقدم | مراقبة تنفيذ الأدوات / الأخطاء |
| المهارات | قائمة المهارات المتاحة (الأدوات) |
| القنوات | إدارة قنوات المراسلة المتصلة |
| الوكلاء | وكلاء الخلفية وإدارة الجلسات |
| التشخيصات | حالة نظام البوابة والسجلات |
| الإعدادات | جميع إعدادات التطبيق |

راجع صفحة كل علامة تبويب للحصول على الميزات التفصيلية.
