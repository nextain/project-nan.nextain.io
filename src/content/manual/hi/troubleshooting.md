सामान्य समस्याएँ और समाधान।

## AI एजेंट प्रतिक्रिया नहीं दे रहा है

1. Settings > AI Settings में API key या Lab कनेक्शन की जाँच करें
2. इंटरनेट कनेक्शन सत्यापित करें
3. Settings > Lab Account में क्रेडिट बैलेंस की जाँच करें
4. सुनिश्चित करें कि मॉडल सही ढंग से कॉन्फ़िगर किया गया है

## आवाज़ काम नहीं कर रही है

1. सुनिश्चित करें कि Settings > Voice में TTS/STT सक्षम है
2. Google API key की आवश्यकता हो सकती है
3. माइक्रोफ़ोन अनुमतियों की जाँच करें (OS सेटिंग्स)
4. सुनिश्चित करें कि ब्राउज़र मीडिया अनुमतियाँ अवरुद्ध नहीं हैं

## क्रेडिट कम हैं

1. Settings > Lab Account में बैलेंस की जाँच करें
2. Lab Dashboard में उपयोग की समीक्षा करें
3. मुफ़्त क्रेडिट मासिक रूप से रीचार्ज होते हैं
4. 100 क्रेडिट के लिए BASIC ($10/माह) में अपग्रेड करें

## उपकरण काम नहीं कर रहे हैं

1. सुनिश्चित करें कि Settings > Tools में "Enable Tools" चालू है
2. जाँच करें कि Gateway चल रहा है
3. सत्यापित करें कि आवश्यक कौशल Skills टैब में सक्षम हैं
4. सामान्य उपयोगकर्ता वातावरण में, आपको Gateway URL/token मैन्युअल रूप से दर्ज करने की आवश्यकता नहीं है
5. Gateway URL/token केवल सेल्फ-होस्टेड/उन्नत सेटअप के लिए जाँच करें

## ऐप धीमा है या जम गया है

1. एक नई बातचीत शुरू करें (+) — लंबी चैट धीमी हो सकती हैं
2. ऐप को रीस्टार्ट करें
3. पूरी तरह से रीसेट करने के लिए Settings > Reset All का उपयोग करें

## मौसम नहीं दिख रहा है

- मौसम कौशल के लिए इंटरनेट कनेक्शन की आवश्यकता होती है
- विशिष्ट रहें: "मुझे सियोल का मौसम बताओ"
- अंग्रेजी शहर के नामों का उपयोग करके देखें (उदाहरण के लिए, "Seoul weather")

## Lab लॉगिन काम नहीं कर रहा है

1. इंटरनेट कनेक्शन की जाँच करें
2. अपने ब्राउज़र में सीधे naia.nextain.io तक पहुँचने का प्रयास करें
3. सत्यापित करें कि आप अपने Google खाते से साइन इन कर सकते हैं
4. ऐप को रीस्टार्ट करें और फिर से प्रयास करें
