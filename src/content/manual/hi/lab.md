## API Keys

![API key प्रबंधन](lab-keys.png)

डेस्कटॉप ऐप्स और बाहरी सेवाओं के लिए API keys प्रबंधित करें:
- **नया key बनाएँ**: एक key नाम दर्ज करें और जनरेट करें
- **key हटाएँ**: अप्रयुक्त keys हटाएँ
- डेस्कटॉप ऐप से कनेक्ट करते समय Keys स्वचालित रूप से जारी किए जाते हैं

## एकीकरण

अपने naia.nextain.io खाते को बाहरी मैसेंजर के साथ लिंक करें ताकि कहीं से भी AI अवतार के साथ चैट कर सकें।

### Discord
- जब आप Discord के साथ साइन इन करते हैं तो स्वचालित रूप से लिंक हो जाता है
- बॉट आमंत्रण लिंक **सेटिंग्स > एकीकरण** पर पाएँ
- अपने सर्वर में @mention या DM के माध्यम से चैट करें
- आपके लैब खाते से क्रेडिट काटे जाएंगे

### Google Chat
- Google के साथ साइन इन करने पर उपलब्ध है
- Workspace ऐप पंजीकरण के बाद Google Chat के माध्यम से चैट करें
- आपके लैब खाते से क्रेडिट काटे जाएंगे
