जब आप पहली बार ऐप लॉन्च करते हैं, तो एक ऑनबोर्डिंग विज़ार्ड दिखाई देता है। शुरुआत करने के दो तरीके हैं।

## Naia OS से शुरू करें (अनुशंसित)

सबसे आसान तरीका — मुफ़्त और किसी API key की आवश्यकता नहीं।

![Onboarding screen — provider selection](onboarding-provider.png)

1. ऑनबोर्डिंग स्क्रीन पर **Naia OS** बटन पर क्लिक करें
2. naia.nextain.io लॉगिन पेज पर एक ब्राउज़र खुलता है
3. अपने **Google account** से साइन इन करें (पंजीकरण स्वचालित है)
4. आपको स्वचालित रूप से डेस्कटॉप ऐप पर वापस रीडायरेक्ट कर दिया जाएगा
5. साइन-अप पर आपको **20 मुफ़्त क्रेडिट** मिलते हैं

## API Key से शुरू करें

आप Google Gemini जैसे प्रोवाइडर्स से एक API key भी दर्ज कर सकते हैं।

![Onboarding screen — API key entry](onboarding-apikey.png)

1. एक प्रोवाइडर चुनें (Gemini, आदि)
2. **Next** पर क्लिक करें
3. अपनी API key दर्ज करें
4. की को टेस्ट करने के लिए **Validate** पर क्लिक करें

## ऑनबोर्डिंग चरण

लैब लॉगिन या API key दर्ज करने के बाद, इन चरणों को पूरा करें:

AI 아바타 को कोई भी नाम दें जो आप चाहें (जैसे, "AI 아바타", "लूना", "कुकी")।

![AI name input](onboarding-agent-name.png)

### 2. आपका नाम
दर्ज करें कि AI 아바타 आपको क्या कहकर बुलाए।

![Your name input](onboarding-user-name.png)
