常见问题与解决方案。

## AI 代理未响应

1. 在“设置”>“AI 设置”中检查 API Key 或 Lab 连接
2. 验证互联网连接
3. 在“设置”>“Lab 账户”中检查积分余额
4. 确保模型配置正确

## 语音功能不工作

1. 确保在“设置”>“语音”中启用 TTS/STT
2. 可能需要一个 Google API Key
3. 检查麦克风权限（操作系统设置）
4. 确保浏览器媒体权限未被阻止

## 积分不足

1. 在“设置”>“Lab 账户”中检查余额
2. 在 Lab 控制面板中查看使用情况
3. 免费积分每月充值
4. 升级到 BASIC ($10/mo) 以获取 100 积分

## 工具不工作

1. 确保在“设置”>“工具”中开启“启用工具”
2. 检查 Gateway 是否正在运行
3. 在“技能”选项卡中验证所需技能是否已启用
4. 在普通用户环境中，您无需手动输入 Gateway URL/token
5. 仅在自托管/高级设置中检查 Gateway URL/token

## 应用运行缓慢或冻结

1. 开始一个新对话 (+) — 长对话可能会导致速度变慢
2. 重启应用
3. 使用“设置”>“重置所有”进行完全重置

## 天气不显示

- 天气技能需要互联网连接
- 具体一点：“告诉我首尔的天气”
- 尝试使用英文城市名称（例如，“Seoul weather”）

## Lab 登录不工作

1. 检查互联网连接
2. 尝试直接在浏览器中访问 naia.nextain.io
3. 验证您可以使用 Google 帐户登录
4. 重启应用并重试
