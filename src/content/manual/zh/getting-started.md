首次启动应用时，会出现一个引导向导。有两种方式可以开始使用。

## 从 Naia OS 开始（推荐）

最简单的方式 — 免费且无需 API Key。

![Onboarding screen — provider selection](onboarding-provider.png)

1. 在引导屏幕上点击 **Naia OS** 按钮
2. 浏览器将打开 naia.nextain.io 登录页面
3. 使用您的 **Google 帐户** 登录（注册是自动的）
4. 您将自动重定向回桌面应用
5. 注册后您将获得 **20 个免费积分**

## 从 API Key 开始

您也可以输入来自 Google Gemini 等提供商的 API Key。

![Onboarding screen — API key entry](onboarding-apikey.png)

1. 选择一个提供商 (Gemini, 等)
2. 点击 **下一步**
3. 输入您的 API Key
4. 点击 **验证** 来测试 Key

## 引导步骤

完成 Lab 登录或 API Key 输入后，请完成以下步骤：

### 1. 为您的 AI 命名
为 AI 아바타 命名，任何您喜欢的名字都可以（例如，“AI 아바타”、“Luna”、“Cookie”）。

![AI name input](onboarding-agent-name.png)

### 2. 您的名字
输入 AI 아바타 应该如何称呼您。

![Your name input](onboarding-user-name.png)

### 3. 选择角色
从 4 个 VRM 头像中选择：

| 角色 | 描述 |
|---|---|
| Shino (深色) | 深色制服 |
| Shino (浅色) | 浅色制服 |
| 女孩 | 普通女孩 |
| 男孩 | 普通男孩 |

![Character selection screen](onboarding-character.png)

### 4. 选择个性
从 4 个预设中选择：

| 个性 | 描述 |
|---|---|
| 友好 | 随意、亲切的语气 |
| 专业 | 礼貌、正式的语气 |
| 活泼 | 幽默、明亮的语气 |
| 学者 | 冷静、理性的语气 |

![Personality selection screen](onboarding-personality.png)

### 5. 即时通讯工具集成（可选）
设置与 Telegram 或 Slack 等支持的即时通讯工具的集成。此步骤可以跳过，稍后在设置中配置。

![Messenger integration screen](onboarding-messenger.png)

### 6. 完成

![Onboarding complete screen](onboarding-complete.png)
点击 **开始使用** 进入主屏幕。
