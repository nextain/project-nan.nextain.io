与 AI 아바타 的主对话屏幕。

## 文本聊天

![聊天屏幕](chat-text.png)

1.  在底部的输入框中输入消息
2.  按下 **Enter** 键或点击 **发送按钮**
3.  AI 아바타 生成回复（实时流式传输）
4.  使用 Shift+Enter 进行换行

## 语音聊天

1.  点击输入框旁边的 **麦克风按钮**
2.  当麦克风激活时讲话
3.  再次点击以停止录音 — 语音转换为文本 (STT)
4.  转换后的文本会自动发送

## 语音回复 (TTS)

-   AI 아바타 的回复会自动以音频形式播放
-   可以在“设置”中关闭
-   播放时，아바타 会进行唇形同步

## 费用显示

![费用仪表盘](chat-cost.png)

-   当前会话的**总 API 费用**显示在右上角
-   点击以打开详细费用仪表盘：
    -   每个提供商的消息数
    -   输入/输出令牌计数
    -   实验室余额（连接时）
    -   充值链接

## 新对话

-   点击右上角的 **+** 按钮开始新对话
-   之前的对话保存在“历史记录”选项卡中

## 工具执行显示

当 AI 아바타 使用工具时，聊天区域显示：

![工具执行显示](chat-tool.png)

-   工具名称（例如，“读取文件”，“执行命令”）
-   执行状态（运行中 / 成功 / 错误）
-   结果（可展开）

## 审批模态框

对于高安全性工具执行：

| 按钮         | 描述             |
| ------------ | ---------------- |
| **允许一次** | 允许本次执行     |
| **始终允许** | 此后自动审批此工具 |
| **拒绝**     | 拒绝执行         |
