主屏幕分为两个区域。

![主屏幕布局](main-screen.png)

## 虚拟形象区域（顶部）

- 您选择的VRM角色以3D形式呈现
- **情绪**根据对话而变化：
  - [HAPPY] — 喜悦
  - [SAD] — 悲伤
  - [ANGRY] — 愤怒
  - [SURPRISED] — 惊讶
  - [NEUTRAL] — 中立
  - [THINK] — 思考
- **摄像头控制**：在虚拟形象区域拖动以移动视角
- **背景图片**：出现在虚拟形象的后面
- **唇形同步**：在TTS响应期间播放动画

## 面板区域（底部）

底部面板有8个选项卡：

| 选项卡 | 描述 |
|---|---|
| 聊天 | 与 AI 아바타 的对话 |
| 历史记录 | 之前的对话 |
| 进度 | 工具执行/错误监控 |
| 技能 | 可用技能（工具）列表 |
| 频道 | 管理已连接的消息频道 |
| 代理 | 后台代理和会话管理 |
| 诊断 | 网关系统健康和日志 |
| 设置 | 所有应用设置 |

有关详细功能，请参阅每个选项卡的页面。
