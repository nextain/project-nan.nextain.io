Common issues and solutions.

## AI agent is not responding

1. Check API key or Lab connection in Settings > AI Settings
2. Verify internet connection
3. Check credit balance in Settings > Lab Account
4. Ensure the model is correctly configured

## Voice is not working

1. Ensure TTS/STT is enabled in Settings > Voice
2. A Google API key may be required
3. Check microphone permissions (OS settings)
4. Ensure browser media permissions are not blocked

## Low on credits

1. Check balance in Settings > Lab Account
2. Review usage in Lab Dashboard
3. Free credits recharge monthly
4. Upgrade to BASIC ($10/mo) for 100 credits

## Tools are not working

1. Ensure "Enable Tools" is on in Settings > Tools
2. Check that Gateway is running
3. Verify needed skills are enabled in Skills tab
4. In normal user environments, you do not need to enter Gateway URL/token manually
5. Check Gateway URL/token only for self-hosted/advanced setups

## Weather is not showing

- The weather skill requires an internet connection
- Be specific: "Tell me the weather in Seoul"
- Try with English city names (e.g., "Seoul weather")

## Lab login is not working

1. Check internet connection
2. Try accessing lab.cafelua.com directly in your browser
3. Verify you can sign in with your Google account
4. Restart the app and try again
