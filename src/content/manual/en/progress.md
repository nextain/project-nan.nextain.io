## Statistics Summary

Four stats at the top:

| Item | Description |
|------|-------------|
| **Total Events** | Total number of operations |
| **Total Cost** | API costs used |
| **Tools Used** | Number of tools invoked |
| **Errors** | Number of errors encountered |

## Event List

- Each event shows type, timestamp, and details
- **Refresh** button for latest data
- Filter to show errors only
