When you first launch the app, an onboarding wizard appears. There are two ways to get started.

## Start with Cafelua Lab (Recommended)

The easiest way — free and no API key needed.

![Onboarding screen — provider selection](onboarding-provider.png)

1. Click the **Cafelua Lab** button on the onboarding screen
2. A browser opens to the lab.cafelua.com login page
3. Sign in with your **Google account** (registration is automatic)
4. You'll be automatically redirected back to the desktop app
5. You receive **20 free credits** on sign-up

## Start with API Key

You can also enter an API key from providers like Google Gemini.

![Onboarding screen — API key entry](onboarding-apikey.png)

1. Select a provider (Gemini, etc.)
2. Click **Next**
3. Enter your API key
4. Click **Validate** to test the key

## Onboarding Steps

After Lab login or API key entry, complete these steps:

### 1. Name Your AI
Give Alpha any name you'd like (e.g., "Alpha", "Luna", "Cookie").

![AI name input](onboarding-agent-name.png)

### 2. Your Name
Enter what Alpha should call you.

![Your name input](onboarding-user-name.png)

### 3. Choose Character
Pick from 4 VRM avatars:

| Character | Description |
|-----------|-------------|
| Shino (Dark) | Dark uniform |
| Shino (Light) | Light uniform |
| Girl | Generic girl |
| Boy | Generic boy |

![Character selection screen](onboarding-character.png)

### 4. Choose Personality
Pick from 4 presets:

| Personality | Description |
|-------------|-------------|
| Friendly | Casual, warm tone |
| Professional | Polite, formal tone |
| Playful | Humorous, bright tone |
| Scholar | Calm, intellectual tone |

![Personality selection screen](onboarding-personality.png)

### 5. Complete

![Onboarding complete screen](onboarding-complete.png)
Click **Get Started** to enter the main screen.
